MOBILE DEVELOPMENT 4201

There is a big demand for mobile development due to the large and medium sized companies are increasingly adapting to the mobile world in this rapidly and technologically changing society. This course investigates the different Mobile Architectures (Native, Web App and Cross-Platform) on various Platforms with examples. The course introduces the students the concepts, tools and techniques used to develop quality Mobile Apps for diverse Platforms and Devices, including Apple iOS, Android, and Windows with examples covering all the different Architectures. The students will create Mobile Apps covering Native and Hybrid Architectures using easily available Tools

Mobile application development tool: Android Studio

1. Tool Name: Android Studio, 3.5 RC 2

2. Platform: Can be downloaded on macOS, Windows and Linux. It runs on all platforms. Android studio iS a hybrid app as it can be used in any of the platform mentioned above.

3. Application Development type:

a. Type: Native type.

b. Advantages: The first advantage is that they provide a maximum performance and doesnot depends on the internet connectivity. The second advantage is that the Native apps are more secure than the web apps

c. Disadvantages: The first disadvantage is that developing the Native App might be very expensive as compared to the web apps. The second advantage is that it doesnot run on every platform as the functionalities changes on every other phone companies.
